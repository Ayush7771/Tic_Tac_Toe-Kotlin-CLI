# Tic Tac Toe (Kotlin)

A simple console-based Tic Tac Toe game built using Kotlin.

## Features

- Two-player gameplay (X vs O)
- Input validation
- Prevents overwriting occupied cells
- Detects winning conditions
- Detects draw condition
- Handles invalid inputs gracefully

## Technologies Used

- Kotlin
- IntelliJ IDEA

## How to Run

1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run the `main()` function

## Example

```
---------
|       |
|       |
|       |
---------

1 1

---------
| X     |
|       |
|       |
---------
```
